# Definition
Innovation Studio is a production from BMC Helix.  
Adapt is a BMC Helix Graphical library, a bit like Google Material, that contains elements like buttons, cards, forms, etc....  
Here is a list of Adapt icons that can be used in your code.

## Icon list
The icons are defined in the file `/bundle/src/main/webapp/node_modules/@bmc-ux/dpl-iconfont/dist/scss/_dpl-iconfont-glyphs.scss`, but this file is extremely long and not very readable, so do not read it.

## How to use an Adapt icon?
### In html templates
To use an Adapt icon in an html template, use its class name.  
To get the class name, simply prefix the icon name with `d-icon-`.  
For example for the icon `widget` the class name will be `d-icon-widget`.
```html
<span class="d-icon-widget"></span>
```

## In a View Component registration module
When registering a View Component, you can use an Adapt icon by specifying its name in the `icon` property of the registration object, for example:
```typescript
icon: 'widget',
```

**Critical**:  
when using an Adapt icon in a View Component registration module, you should only specify the icon name, without the `d-icon-` prefix, otherwise the icon will not be displayed and you will not have any error message to understand why, so be careful when choosing the icon name. Only use the icon name as it is in the list above, else no icon will be displayed in the Innovation Studio View Designer.
